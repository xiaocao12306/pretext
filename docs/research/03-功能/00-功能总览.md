# 功能总览

本文档汇总“Pretext 提供了哪些能力”，后续会拆成更细的功能卡。

## 核心功能
- 高度预测：`prepare()` + `layout()`
- 富行布局：`prepareWithSegments()` + `layoutWithLines()` + `walkLineRanges()` + `layoutNextLine()`
- locale 与 cache 管理：`setLocale()`、`clearCache()`
- whitespace 模式：默认 `normal` 与显式 `{ whiteSpace: 'pre-wrap' }`

## 工程功能
- 浏览器准确性校验
- benchmark 与性能快照
- corpus / Gatsby / probe 诊断
- demos 与静态站点构建
- package tarball smoke test
