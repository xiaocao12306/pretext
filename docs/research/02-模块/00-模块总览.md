# 模块总览

本文档承接 [[01-项目总览]]，后续将拆成更细的模块卡。

## 核心模块簇
- `src/layout.ts`：公共 API 编排层
- `src/analysis.ts`：文本分析与预处理层
- `src/measurement.ts`：canvas 度量与缓存层
- `src/line-break.ts`：断行执行层
- `src/bidi.ts`：rich path bidi 元数据层

## 支撑模块簇
- `pages/`：浏览器验证、可视化诊断、demo dogfooding
- `scripts/`：自动化校验、快照生成、报告通道
- `corpora/`：长文 canary 与研究数据
